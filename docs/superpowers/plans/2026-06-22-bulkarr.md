# Bulkarr Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Next.js 14 web app that bulk-adds movies to Radarr and TV series to Sonarr from a pasted title list, with match review, per-row overrides, and full state persistence across refreshes and container restarts.

**Architecture:** Single-page app with four domain hooks (useSettings, useSession, useLookup, useSubmit) owning their state slices and persisting to a server-side JSON store via Next.js route handlers. All *arr API calls proxy through route handlers — API keys never reach the browser.

**Tech Stack:** Next.js 14 App Router, TypeScript, Tailwind CSS, pnpm, Jest + @testing-library/react, @tanstack/react-virtual, uuid, Docker standalone output

import fs from 'fs'
import path from 'path'
import type { Store } from './types'

const DEFAULT_STORE: Store = {
  settings: { radarr: null, sonarr: null },
  cache: { radarr: null, sonarr: null },
  sessions: { movies: null, series: null },
}

function getStorePath(): string {
  return path.join(process.env.DATA_DIR ?? './data', 'store.json')
}

export function readStore(): Store {
  try {
    const raw = fs.readFileSync(getStorePath(), 'utf-8')
    return { ...structuredClone(DEFAULT_STORE), ...JSON.parse(raw) }
  } catch {
    return structuredClone(DEFAULT_STORE)
  }
}

export function writeStore(store: Store): void {
  const p = getStorePath()
  fs.mkdirSync(path.dirname(p), { recursive: true })
  fs.writeFileSync(p, JSON.stringify(store, null, 2), 'utf-8')
}

export function updateStore(updater: (store: Store) => void): Store {
  const store = readStore()
  updater(store)
  writeStore(store)
  return store
}
```

export async function throttledBatch<T, R>(
  items: T[],
  fn: (item: T, index: number) => Promise<R>,
  { concurrency = 3, delayMs = 300 }: { concurrency?: number; delayMs?: number } = {}
): Promise<R[]> {
  if (!items.length) return []
  const results: R[] = new Array(items.length)
  let next = 0
  let lastDispatch = 0

  async function worker() {
    while (next < items.length) {
      const i = next++
      if (delayMs > 0) {
        const wait = lastDispatch + delayMs - Date.now()
        if (wait > 0) await new Promise(r => setTimeout(r, wait))
      }
      lastDispatch = Date.now()
      results[i] = await fn(items[i], i)
    }
  }

  await Promise.all(Array.from({ length: Math.min(concurrency, items.length) }, worker))
  return results
}
```

import type { QualityProfile, RootFolder, LangProfile, LibraryItem, ArrItem, ArrErrorCode } from './types'

export class ArrApiError extends Error {
  code: ArrErrorCode
  httpStatus?: number
  constructor(code: ArrErrorCode, message: string, httpStatus?: number) {
    super(message)
    this.code = code
    this.httpStatus = httpStatus
  }
}

async function arrFetch(baseUrl: string, apiKey: string, path: string, init: RequestInit = {}): Promise<unknown> {
  const url = `${baseUrl.replace(/\/+$/, '')}${path}`
  let res: Response
  try {
    res = await fetch(url, {
      ...init,
      headers: { 'X-Api-Key': apiKey, 'Content-Type': 'application/json', ...init.headers },
    })
  } catch (e) {
    throw new ArrApiError('UNREACHABLE', `Cannot reach ${baseUrl}: ${(e as Error).message}`)
  }
  if (res.status === 401) throw new ArrApiError('AUTH_FAILED', 'Invalid API key', 401)
  if (res.status === 400) {
    const b = await res.json().catch(() => ({})) as { message?: string }
    throw new ArrApiError('BAD_REQUEST', b.message ?? 'Bad request', 400)
  }
  if (res.status === 404) throw new ArrApiError('NOT_FOUND', 'Not found', 404)
  if (!res.ok) throw new ArrApiError('UNKNOWN', `HTTP ${res.status}`, res.status)
  return res.json()
}

export async function getSystemStatus(url: string, key: string) {
  return arrFetch(url, key, '/api/v3/system/status') as Promise<{ version: string }>
}

export async function getQualityProfiles(url: string, key: string): Promise<QualityProfile[]> {
  return arrFetch(url, key, '/api/v3/qualityprofile') as Promise<QualityProfile[]>
}

export async function getRootFolders(url: string, key: string): Promise<RootFolder[]> {
  return arrFetch(url, key, '/api/v3/rootfolder') as Promise<RootFolder[]>
}

export async function getLangProfiles(url: string, key: string): Promise<LangProfile[]> {
  try { return await arrFetch(url, key, '/api/v3/languageprofile') as LangProfile[] }
  catch { return [] }
}

export async function getMovieLibrary(url: string, key: string): Promise<LibraryItem[]> {
  const items = await arrFetch(url, key, '/api/v3/movie') as Array<{ id: number; title: string; tmdbId: number }>

}

export const runtime = 'nodejs'

function maskKey(k: string): string {
  if (!k) return ''
  return k.length <= 8 ? '••••••••' : `${k.slice(0, 4)}${'•'.repeat(k.length - 4)}`
}

function seedFromEnv(store: ReturnType<typeof readStore>) {
  if (!store.settings.radarr && process.env.RADARR_URL && process.env.RADARR_API_KEY)
    store.settings.radarr = { url: process.env.RADARR_URL, apiKey: process.env.RADARR_API_KEY }
  if (!store.settings.sonarr && process.env.SONARR_URL && process.env.SONARR_API_KEY)
    store.settings.sonarr = { url: process.env.SONARR_URL, apiKey: process.env.SONARR_API_KEY }

import { useState, useCallback } from 'react'
import { v4 as uuidv4 } from 'uuid'
import type { ReviewRow, Target, LibraryItem, ArrItem } from '@/lib/types'

export function useLookup() {
  const [progress, setProgress] = useState<{ done: number; total: number } | null>(null)
  const [looking, setLooking] = useState(false)

  const lookup = useCallback(async (
    terms: string[],
    target: Target,
    library: LibraryItem[],
    onRows: (rows: ReviewRow[]) => void
  ) => {
    if (!terms.length) return
    setLooking(true)
    setProgress({ done: 0, total: terms.length })

    try {
      const r = await fetch('/api/lookup', {
        method: 'POST', headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ target, terms }),
      })
      if (!r.ok) { const e = await r.json(); throw new Error(e?.error?.message ?? 'Lookup failed') }

      const { results } = await r.json() as { results: { candidates: ArrItem[]; error?: string }[] }
      const libSet = new Set(library.map(i => target === 'movies' ? i.tmdbId : i.tvdbId))

      const rows: ReviewRow[] = terms.map((inputText, i) => {
        const { candidates = [], error } = results[i] ?? {}
        const top = candidates[0]
        const inLib = !!top && libSet.has(target === 'movies' ? top.tmdbId : top.tvdbId)
        return {
          id: uuidv4(), inputText, candidates, selectedIndex: 0, overrides: {},
          included: candidates.length > 0 && !inLib,
          status: inLib ? 'in_library' : candidates.length ? 'matched' : 'no_match',
          errorMessage: error,
        }
      })

      setProgress({ done: terms.length, total: terms.length })
      onRows(rows)
    } finally {
      setLooking(false)
      setTimeout(() => setProgress(null), 1500)
    }
  }, [])

  return { lookup, progress, looking }
}

import { useState, useCallback } from 'react'
import type { ReviewRow, DefaultsConfig, Target, SubmitResult } from '@/lib/types'

export interface SubmitSummary { added: number; failed: number; skipped: number }

export function useSubmit() {
  const [submitting, setSubmitting] = useState(false)
  const [summary, setSummary] = useState<SubmitSummary | null>(null)

  const submit = useCallback(async (
    rows: ReviewRow[], target: Target, defaults: DefaultsConfig,
    onUpdateRow: (id: string, patch: Partial<ReviewRow>) => void
  ) => {
    const eligible = rows.filter(r => r.included && r.status === 'matched')
    if (!eligible.length) return
    setSubmitting(true); setSummary(null)

    try {
      const r = await fetch('/api/submit', {
        method: 'POST', headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ target, rows: eligible, defaults }),
      })
      if (!r.ok) { const e = await r.json(); throw new Error(e?.error?.message ?? 'Submit failed') }

      const { results } = await r.json() as { results: SubmitResult[] }
      let added = 0, failed = 0

      for (const res of results) {
        if (res.status === 'added') { added++; onUpdateRow(res.rowId, { status: 'added', included: false }) }
        else { failed++; onUpdateRow(res.rowId, { status: 'failed', errorMessage: res.errorMessage }) }
      }

      setSummary({ added, failed, skipped: rows.length - eligible.length })
    } finally {
      setSubmitting(false)
    }
  }, [])

  return { submit, submitting, summary }
}

'use client'
import { useRef } from 'react'
import { useVirtualizer } from '@tanstack/react-virtual'
import type { ReviewRow, ServiceCache } from '@/lib/types'
import { ReviewRowComponent } from './ReviewRow'

function THead() {
  return (
    <thead className="sticky top-0 z-10 bg-slate-900">
      <tr className="border-b border-slate-700 text-xs text-slate-500 uppercase tracking-wider">
        <th className="px-3 py-2 w-8 text-left"></th>
        <th className="px-2 py-2 w-36 text-left">Input</th>
        <th className="px-2 py-2 text-left">Match</th>
        <th className="px-2 py-2 w-24 text-left">Status</th>
        <th className="px-2 py-2 w-36 text-left">Overrides</th>
        <th className="px-2 py-2 w-14 text-left"></th>
      </tr>
    </thead>
  )
}

export function ReviewTable({ rows, cache, onUpdateRow, onResearch }: {
  rows: ReviewRow[]
  cache: ServiceCache | null | undefined
  onUpdateRow: (id: string, p: Partial<ReviewRow>) => void
  onResearch: (id: string) => void
}) {
  const parentRef = useRef<HTMLDivElement>(null)
  const virt = useVirtualizer({
    count: rows.length,
    getScrollElement: () => parentRef.current,
    estimateSize: () => 64,
    overscan: 5,
    enabled: rows.length > 100,
  })

  const virtItems = virt.getVirtualItems()
  const paddingTop = virtItems.length > 0 ? virtItems[0].start : 0
  const paddingBottom = virtItems.length > 0 ? virt.getTotalSize() - virtItems[virtItems.length - 1].end : 0
  const displayRows = rows.length > 100 ? virtItems.map(vi => rows[vi.index]) : rows

  return (
    <div ref={parentRef} className="overflow-auto" style={{ maxHeight: 'calc(100vh - 280px)' }}>
      <table className="w-full text-left table-fixed">
        <THead />
        <tbody>
          {rows.length > 100 && paddingTop > 0 && <tr><td style={{ height: paddingTop }} /></tr>}
          {displayRows.map(row => (
            <ReviewRowComponent
              key={row.id}
              row={row}
              cache={cache}
              onUpdate={p => onUpdateRow(row.id, p)}
              onDelete={() => onUpdateRow(row.id, { included: false, status: 'no_match' })}
              onResearch={() => onResearch(row.id)}
            />
          ))}
          {rows.length > 100 && paddingBottom > 0 && <tr><td style={{ height: paddingBottom }} /></tr>}
        </tbody>
      </table>
    </div>
  )
}
```

'use client'
import { useState, useCallback } from 'react'
import { useSettings } from '@/hooks/useSettings'
import { useSession } from '@/hooks/useSession'
import { useLookup } from '@/hooks/useLookup'
import { useSubmit } from '@/hooks/useSubmit'
import { useToast } from '@/hooks/useToast'
import { SettingsDrawer } from '@/components/SettingsDrawer'
import { DefaultsBar } from '@/components/DefaultsBar'
import { InputPanel } from '@/components/InputPanel'
import { ReviewTable } from '@/components/ReviewTable'
import { ToastStack } from '@/components/ToastStack'

export default function Home() {
  const [settingsOpen, setSettingsOpen] = useState(false)
  const { settings, cache, saveSettings, testConnection, refreshCache, refetch } = useSettings()
  const session = useSession()
  const { lookup, progress, looking } = useLookup()
  const { submit, submitting, summary } = useSubmit()
  const { toasts, addToast, removeToast } = useToast()

  const activeCache = session.target === 'movies' ? cache.radarr : cache.sonarr

  const handleLookup = async () => {
    const terms = session.parseLines()
    if (!terms.length) return
    try {
      await lookup(terms, session.target, activeCache?.library ?? [], session.setRows)
    } catch (e) {
      addToast((e as Error).message, 'error')
    }
  }

  const handleResearch = useCallback(async (rowId: string) => {
    const row = session.rows.find(r => r.id === rowId)
    if (!row) return
    try {
      await lookup([row.inputText], session.target, activeCache?.library ?? [], newRows => {
        if (newRows[0]) session.updateRow(rowId, {
          candidates: newRows[0].candidates,
          selectedIndex: 0,
          status: newRows[0].status,
          included: newRows[0].included,
          errorMessage: newRows[0].errorMessage,
        })
      })
    } catch (e) {
      addToast((e as Error).message, 'error')
    }
  }, [session, lookup, activeCache, addToast])

  const handleSubmit = async () => {
    try {
      await submit(session.rows, session.target, session.defaults, session.updateRow)
    } catch (e) {
      addToast((e as Error).message, 'error')
    }
  }

  const selectedCount = session.rows.filter(r => r.included && r.status === 'matched').length

  return (
    <div className="flex flex-col min-h-screen">
      {/* Top bar */}
      <header className="flex items-center justify-between px-4 py-2 border-b border-slate-700 bg-slate-800 shrink-0">
        <h1 className="text-orange-500 font-bold tracking-tight">Bulkarr</h1>
        <div className="flex items-center gap-2">
          <button onClick={() => refetch().then(() => addToast('Refreshed', 'info')).catch(() => {})}
            className="text-slate-400 hover:text-slate-200 text-sm px-2 py-1 rounded hover:bg-slate-700">
            ↻
          </button>
          <button onClick={() => setSettingsOpen(true)}
            className="text-slate-400 hover:text-slate-200 text-sm px-2 py-1 rounded hover:bg-slate-700">
            ⚙ Settings
          </button>
        </div>
      </header>

      <DefaultsBar
        target={session.target}
        onTargetChange={session.setTarget}
        defaults={session.defaults}
        onDefaultsChange={session.setDefaults}
        cache={activeCache}
      />

      <InputPanel
        value={session.rawInput}
        onChange={session.setRawInput}
        onLookup={handleLookup}
        looking={looking}
        progress={progress}
      />

      {session.rows.length > 0 && (
        <div className="flex-1 min-h-0">
          <ReviewTable
            rows={session.rows}
            cache={activeCache}
            onUpdateRow={session.updateRow}
            onResearch={handleResearch}
          />
        </div>
      )}

      {/* Submit bar */}
      {session.rows.length > 0 && (
        <footer className="shrink-0 sticky bottom-0 border-t border-slate-700 bg-slate-800 px-4 py-2 flex items-center justify-between gap-4">
          <div className="flex items-center gap-4">
            <button onClick={session.clearSession}
              className="text-xs text-slate-500 hover:text-red-400">
              Clear Session
            </button>
            {summary && (
              <span className="text-xs text-slate-300">
                {summary.added} added · {summary.failed} failed · {summary.skipped} skipped
              </span>
            )}
          </div>
          <button onClick={handleSubmit} disabled={submitting || selectedCount === 0}
            className="bg-orange-500 hover:bg-orange-400 disabled:opacity-40 disabled:cursor-not-allowed text-white font-semibold px-5 py-2 rounded text-sm">
            {submitting ? 'Adding…' : `Add Selected (${selectedCount})`}
          </button>
        </footer>
      )}

      <SettingsDrawer
        open={settingsOpen}
        onClose={() => setSettingsOpen(false)}
        settings={settings}
        onSave={saveSettings}
        onTest={testConnection}
        onRefreshCache={refreshCache}
        addToast={addToast}
      />

      <ToastStack toasts={toasts} onRemove={removeToast} />
    </div>
  )
}
```

Bulk-add movies to **Radarr** and TV series to **Sonarr** from a pasted list of titles.

## Docker Deployment

### 1. Configure secrets

```bash
cp .env.example .env
# Edit .env — fill in RADARR_URL, RADARR_API_KEY, SONARR_URL, SONARR_API_KEY
# You can also leave them blank and configure via the Settings panel
```

### 2. Start

```bash
docker compose up -d --build
```

The app is available at http://your-host:3000.

### 3. Where data lives

Config, API keys, and in-progress sessions are stored at `DATA_DIR` inside the container (`/app/data` by default), backed by the `bulkarr_data` named volume.  
Keys survive `docker compose down` and `docker compose up` because the volume is not removed unless you run `docker volume rm`.

To inspect or back up the store:
```bash
docker compose exec bulkarr cat /app/data/store.json
```

### 4. Join your existing media network

If Radarr/Sonarr run in their own Docker stack (e.g. on network `media`), you can reach them by container name instead of LAN IP.

In `docker-compose.yml`, uncomment:
```yaml
networks:
  default:
    name: media
    external: true
```

Then in `.env`:
```
RADARR_URL=http://radarr:7878
SONARR_URL=http://sonarr:8989
```

### 5. Reverse proxy / URL base path

If Radarr or Sonarr sit behind a reverse proxy at a sub-path (e.g. `/radarr`), include that path in the base URL:

```
RADARR_URL=http://192.168.1.50/radarr
```

The client prepends it to every API call automatically.

## Environment Variables

| Variable | Default | Description |
|---|---|---|
| `RADARR_URL` | — | Radarr base URL (seeds config on first boot) |
| `RADARR_API_KEY` | — | Radarr API key (seeds config on first boot) |
| `SONARR_URL` | — | Sonarr base URL |
| `SONARR_API_KEY` | — | Sonarr API key |
| `DATA_DIR` | `./data` | Directory for `store.json` |
| `PORT` | `3000` | Listen port |

Env vars are **seed-only**: if the Settings panel has already saved config, env vars are ignored on restart.
